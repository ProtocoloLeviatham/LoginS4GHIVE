VOLVER AL ARCENAL

INTRODUCCIÓN AL SISTEMA

El **LEVIATHAM S.E.T ARCENAL** es una suite modular diseñada para pruebas de penetración éticas, auditoría de seguridad y análisis forense. Esta herramienta centraliza capacidades de reconocimiento (OSINT), militarización (Weaponization) y control, conectándose a una base de datos global en tiempo real.

ADVERTENCIA: El uso de estas herramientas contra sistemas sin autorización expresa es ilegal. Utilice este entorno únicamente con fines educativos o auditorías autorizadas.

SECCIÓN 1: RECONOCIMIENTO (RECON)

OSINT EMAIL HARVESTER V3

Motor de recolección de inteligencia de fuentes abiertas enfocado en identidades corporativas.

Funcionamiento: Simula la búsqueda en múltiples bases de datos públicas (motores de búsqueda, repositorios de claves PGP, fugas de datos) para encontrar patrones de correo electrónico asociados a un dominio.

Caso de Uso Real:
Durante una auditoría de ingeniería social, un pentester necesita saber el formato de correo de la empresa (ej: `nombre.apellido@empresa.com`) para simular una campaña de phishing autorizada y medir la concientización de los empleados.

WEB RECON EXTRACTOR WEB (LINK SCRAPER)

Analizador de superficie de ataque web.

Funcionamiento: Realiza una petición HTTP a la URL objetivo y parsea el DOM para extraer todos los hipervínculos, fuentes de imágenes y scripts. Utiliza un proxy CORS para evitar bloqueos de navegador.

Caso de Uso Real:
Un bug bounty hunter utiliza esta herramienta para encontrar archivos JavaScript (`.js`) ocultos o de desarrollo que los desarrolladores olvidaron quitar, los cuales a menudo contienen claves API o rutas a paneles de administración no documentados.

INFRA DETECTOR DE WAF

Identificación de Firewalls de Aplicación Web.

Funcionamiento: Analiza las cabeceras de respuesta HTTP y cookies específicas (como `cf-ray` de Cloudflare o `x-iinfo` de Imperva) para determinar qué tecnología protege al servidor.

Caso de Uso Real:
Antes de lanzar un escáner de vulnerabilidades (como Nessus o SQLMap), el auditor necesita saber si hay un WAF activo para ajustar la velocidad del escaneo o utilizar técnicas de evasión (bypass) para no ser bloqueado inmediatamente.

OPSEC DESACORTADOR URL

Análisis seguro de enlaces ofuscados.

Funcionamiento: Sigue la cadena de redirecciones HTTP (301/302) de servicios como bit.ly o tinyurl y muestra el destino final sin cargar el contenido malicioso en el navegador.

Caso de Uso Real:
Un analista de SOC recibe un reporte de un empleado sobre un SMS sospechoso con un enlace corto. Usa esta herramienta para ver a dónde lleva el enlace sin infectar su propia máquina de análisis.

HARDENING ANALIZADOR DE CABECERAS

Auditoría de configuración de seguridad del servidor.

Funcionamiento: Verifica la presencia de headers críticos como `Content-Security-Policy` (CSP), `X-Frame-Options` y `Strict-Transport-Security` (HSTS).

Caso de Uso Real:
Un administrador de sistemas utiliza esta herramienta después de desplegar una nueva web para asegurarse de que el servidor está configurado para rechazar ataques de Clickjacking y forzar conexiones HTTPS seguras.

SECCIÓN 2: WEAPONIZATION (ARMAMENTO)

BRUTE FORCE GENERADOR DICCIONARIOS PRO

Creación de wordlists personalizadas contextuales.

Funcionamiento: Motor de permutación que combina datos específicos del objetivo (nombre, mascota, año) con patrones comunes de contraseñas. Incluye modos de duración variable para generar listas masivas.

Caso de Uso Real:
En una auditoría interna, se descubre que la política de contraseñas es débil. El auditor genera un diccionario basado en el nombre de la empresa y el año actual (ej: `Empresa2024!`) para demostrar cuán fácil es adivinar las credenciales de los empleados.

INJECTION GENERADOR SQLi

Vectores de prueba para bases de datos SQL.

Funcionamiento: Provee una lista curada de payloads SQL Injection clásicos y modernos para probar bypass de autenticación y extracción de datos.

Caso de Uso Real:
Un pentester encuentra un formulario de login antiguo. Prueba el payload `' OR '1'='1` para ver si el servidor valida incorrectamente la entrada y le permite entrar como administrador sin contraseña.

NETWORK FABRICANTE MAC (OUI)

Identificación de hardware de red.

Funcionamiento: Consulta la base de datos IEEE OUI para identificar al fabricante de un dispositivo basándose en los primeros 6 caracteres de su dirección MAC.

Caso de Uso Real:
Durante un escaneo de red WiFi, se detecta un dispositivo desconocido. Al buscar su MAC, la herramienta revela que es un "Espressif Inc", lo que indica que probablemente sea un dispositivo IoT o una cámara inteligente barata, a menudo vulnerables.

NETWORK CALCULADORA DE SUBRED

Planificación de alcance de red.

Funcionamiento: Convierte notación CIDR (ej: /24) en máscaras de subred y calcula la cantidad de hosts útiles.

Caso de Uso Real:
Un auditor necesita escanear toda la red interna de un cliente. El cliente le da el rango `10.0.0.0/19`. Usando la calculadora, el auditor sabe exactamente cuántas IPs (8190 hosts) debe configurar en su herramienta de escaneo.

SECCIÓN 3: CONTROL & BLUE TEAM

CRYPTO ENCODER / DECODER

Manipulación de datos en tiempo real.

Funcionamiento: Transforma texto plano a formatos comunes (Base64, Hex, Binario, URL) y viceversa.

Caso de Uso Real:
Al analizar una URL sospechosa, un analista ve un parámetro `?data=YWRtaW4=`. Usa el decodificador Base64 para revelar que dice "admin", confirmando un intento de manipulación de parámetros.

CRYPTO CRIPTOANÁLISIS (FRECUENCIA)

Rompedor de cifrados clásicos.

Funcionamiento: Analiza la frecuencia de aparición de cada letra en un texto cifrado. Útil para ataques estadísticos contra cifrados de sustitución monoalfabética.

Caso de Uso Real:
En un ejercicio CTF (Capture The Flag), se proporciona un mensaje ininteligible. El análisis de frecuencia muestra que la letra 'X' es la más común, sugiriendo que 'X' corresponde a la 'E' (la letra más común en español/inglés), dando pie a descifrar el mensaje.

EXPLOIT GENERADOR XSS PERSONALIZABLE

Gestión de vectores Cross-Site Scripting.

Funcionamiento: Permite almacenar, editar y copiar rápidamente scripts complejos para pruebas XSS. Conectado al Panel de Admin para persistencia.

Caso de Uso Real:
Un investigador descubre un filtro XSS que bloquea la palabra "script". Crea un payload personalizado `` y lo guarda en el generador para reusarlo rápidamente en diferentes campos del formulario vulnerable.

MALWARE REPOSITORIO DE AMENAZAS

Base de conocimiento de firmas y scripts.

Funcionamiento: Biblioteca editable de fragmentos de código malicioso conocidos para estudio o simulación de ataques.

Caso de Uso Real:
Un Blue Teamer quiere configurar su IDS (Sistema de Detección de Intrusos) para detectar el gusano "ILoveYou". Copia el código VBS del repositorio para estudiar su firma y crear una regla de detección precisa.

FORENSICS ESTEGANOGRAFÍA (STEGO)

Ocultación de información.

Funcionamiento: Utiliza caracteres invisibles (Zero Width Characters) para ocultar un mensaje secreto dentro de un texto normal visible.

Caso de Uso Real:
Validación de DLP (Data Loss Prevention). Un auditor intenta exfiltrar un "secreto" fuera de la red corporativa ocultándolo en un correo de "Lista de compras" para ver si los sistemas de seguridad detectan la fuga de información oculta.

PRIVACY HUELLA DIGITAL (FINGERPRINT)

Auditoría de privacidad del navegador.

Funcionamiento: Recopila datos pasivos del navegador (User Agent, Resolución, Hardware, Zona Horaria) para mostrar cuán único es el usuario.

Caso de Uso Real:
Demostración de privacidad a un cliente. Se le muestra que, incluso usando el modo incógnito, su combinación específica de hardware y configuración lo hace identificable entre millones de usuarios.

CRACKING GENERADOR LEET (MUTATOR)

Fortalecimiento y mutación de claves.

Funcionamiento: Aplica sustituciones "Leet Speak" (a->4, e->3), alterna mayúsculas y añade "salts" aleatorios para crear variantes complejas de una palabra base.

Caso de Uso Real:
Un usuario quiere una contraseña segura pero recordable basada en "dragon". El mutador genera `dr4G0n_x9Z!`, aumentando exponencialmente la entropía y resistencia a fuerza bruta.

DISCOVERY API FUZZER (AUTO-REDIRECT)

Descubrimiento de endpoints ocultos.

Funcionamiento: Genera enlaces secuenciales o basados en listas sobre una URL base y permite abrirlos masivamente para probar su existencia.

Caso de Uso Real:
Se encuentra una API en `api/v1/user/100`. El auditor usa el Fuzzer con el rango 1-50. Al abrir los enlaces, descubre que el usuario `api/v1/user/1` es el administrador y la API no tiene control de acceso (IDOR), permitiendo ver sus datos.

LOGGING DB GLOBAL

Registro de auditoría inmutable.

Funcionamiento: Conexión en tiempo real a Firebase. Cada acción realizada en el arsenal queda registrada con Timestamp, Herramienta utilizada y la IP pública del operador (parcialmente ofuscada por privacidad).

Valor:
Garantiza la trazabilidad de las acciones durante un ejercicio de Red Team compartido, permitiendo a los supervisores revisar qué herramientas se usaron y cuándo.
